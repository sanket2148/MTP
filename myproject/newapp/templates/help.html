Select User Type: Open the software and choose your user type from the following options: Admin, Annotator, Researcher, or Validator. As an admin, select "Admin".
Enter User Information: Fill in your personal details such as first name, last name, and a user ID, which will be visible to other users in the system.
Create a Password: Create a strong and memorable password for system access.
Note: If registration is done under the incorrect user type, delete the account from settings and create a new one with the correct designation.

Check Your Email: Check your registered email for a verification One-Time Password (OTP) after registration.
Enter OTP for Verification: Input the OTP in the designated field within the software to verify your account and finalize the registration process.

Select User Type: At the login screen, select your user type ("Admin") from the available options.
Enter Login Credentials: Use your registered user ID and password to log in.

Overview: Upon logging in, users are directed to the homepage, which features a structured and user-friendly layout consisting of a navigation menu on the left side and multiple language folders displayed alongside a detailed table.
Language Folders Subsection: The homepage displays various folders organized by language. Each folder contains specific resources and settings pertinent to that language, including files and documentation relevant to tasks within the system. Users can click on a language folder to upload images (up to 99 images at a time), manage image confidentiality, and view or alter the images’ download and deletion settings. A table on the homepage provides details about the folders, including the number of images present and selected.

Select Language: Use the dropdown menu to select the language.
View Assigned Images: View the images assigned to you for annotation for the selected language.
Select Image:Select a image from the image gallery, user navigation buttons( Previous, Next) for navigate through images.
Start Annotation: Press the 'Start' button to begin speaking and annotating images. Annotations are created as you speak, by converting you spokend words into text(Annotation).
Stop and Submit: Press the "Stop" button to stop the annotation , and Press the "Submit Annotation" button to save you annotation.(you can check you annotation in the annotaitons table below).
Annotation Table: This table contains all the annotations you have done preiously for the selected langauge.You can edit of delete any og the annotation by clicking on the "Edit" and "Delete" buttons present in each row , accordingly.

Select Language: Use the dropdown menu to select the language.
Selecting Annotators: Drag and Drop the annotators name to whom you want to assign the annotation task for the selected langauge. Save: After selecting all the annotators you want for all languages , always press "Save Selection" button present above the language table, to save your selections.

Select Language: Use the dropdown menu to select the language.After selecting , List of annotator you selected in the "Add / Remove" will be displayed in a table.
Assigning Images :
Method 1: Drag and Drop the Images from gallery and drop in the annotators table , in the row of annotator to whom you want to assign the annotation task of that image.
Method 2: select the annotators name from the dropdoen to whom yo want to assign the annotation task and then enter the number of images you want to asssign(<= total number of images) adn press the "Randomly Assign" button. Save: After assigning images ,press "Save Assignment" button to save your assignments.

After selecting , List of images will be displayed for the selected language , once you click on any image you will find the annotation of that image in a table. You can also check the annotations according to the annotators , by click on the annotators name , after selecting the "by annotator" radio button. 2) validation :

After selecting the langauge you get the option to assign the validation to the avalibale validators, by selctin the name of the validator from the dropdown and entering the number of annotations to be assigned fo that validator. After selecting the language, list of images will be displayed for the selected language. Once you click on any image, the image will be previewed in the preview box , and you can annotation that image by clicking the "start" button.Once the annotation task is done you have to clcik on the submit the annotation to save it. Select User Type: Open the software and begin by selecting your user type. Choose "Annotator" for annotators like you.
Enter User Information: Fill in your personal details, including your first name, last name, and user ID. The user ID will be visible to other users, so choose one that you are comfortable sharing.
Create a Password: Create a secure password that you will use to log in to the system. Ensure it is strong and memorable.
Note: If you accidentally register under the wrong user type, go to the settings menu, delete your account, and create a new one with the correct user type.

Check your email: After registration, check your registered email for a verification OTP.
Enter OTP for Verification: Enter the OTP in the designated field on the software to verify your account and complete the registration process.

Select User Type: On the login screen, again select your user type from the options provided.
Enter Login Credentials: Log in using your registered user ID and password.

Access Settings: Once logged in, navigate to the settings menu.
Select Preferred Language: Choose the language you prefer for performing annotations or validations.

Access Start Annotation Page: Navigate to the 'Start Annotation' page from the main menu.
Select Language: Use the drop-down menu to select the language you set up previously.
View Assigned Images: The images assigned to you for annotation will be displayed here.

Start Annotation: Press the 'Start' button to begin speaking and annotating the images. Your annotations will be recorded as you speak.
Submit Annotations: After completing the annotations, press 'Submit' to save your work to the database.

Edit Annotations: If you need to modify any annotations, use the 'Edit' option to make changes and add the revised version.
Delete Annotations: If you wish to remove any previous annotations completely, use the 'Delete' option.

Go to the “check annotation” page: Select the language. Click on the images, and you will find the annotations you have made on that image. If there are no annotations, an empty table will be displayed.

Return to Settings: After completing your annotations, go back to the settings menu.
